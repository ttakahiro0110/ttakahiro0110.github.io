How to create your very first website on GitHub?


First, let’s set up GitHub and create your repository:

  1. Go to github.com.
  2. Sign up for a GitHub account (if you have one, log in to your existing account).
  3. Once you are logged in, click your profile icon on the top right and click Your repositories from the drop-down.
  4. Click New button to create a new repository
  5. Name the new repository [your username].github.io and click Create repository green button (leave it as Public for now).
  6. Once your repository is created, you are ready to copy some files into it and upload contents to your very first website.

Now, let’s create some content for your first website:

  1. Open Visual Studio Code (VS) on your computer (if you don’t have it, download it from https://code.visualstudio.com/
  2. In VS, click the Open Folder button and select/create a folder on your computer where you want to save your files.
  3. Go to File > New File, then you will see an untitled new file.
  4. Save the new file as index.html in the folder.
  5. Find a photo you want on your website and upload the photo to the folder you save the html file.
  6. Type sample html codes below and make sure you have the correct image file name in the code (make sure you save the html file).

Lastly, let’s upload the contents to GitHub and set up your very first website!

  1. Go back to Your repository on GitHub.
  2. Click Add file button and select Upload files
  3. Drag the index.html and image files that you created into your repository.
  4. Make sure to click Commit changes button on the bottom.
  5. Now, your very first website is ready. Go to  [your username].github.io on your web browser. You should see the contents you created.
